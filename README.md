# mall-system

> A Vue.js project
（基于vue全家桶，vux框架的移动端商城管理平台）
## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

main入口文件路由跳转需注释才可正常跳转到主页

```
